# A Study of Shape Interpolation Using Optimal Transport 

Graduate thesis at Ho Chi Minh city University of Technology


### Description
Generating smooth transitions between two shapes using optimal transport theory and low-dimensional representation from variational autoencoder. For more details, you can read our thesis [here](https://github.com/LekiizRaii/shape-interpolation-with-optimal-transport/blob/main/assets/CO4029_HK241_OT_final_thesis_report.pdf)

### Tools
Python, PyTorch, 3D data processing, variational autoencoder, optimal transport
